Single Bot Redirect Page

Instructions:
1. Replace 'SIMBHA_NGY_BOT' in index.html with your Telegram bot username.
2. Host the folder online using GitHub Pages, Cloudflare Pages, or any static hosting.
3. Access your page with the 'start' parameter:
   Example: https://yourdomain.com/?start=<base64_encoded_string>
4. The page will decode the parameter and redirect to your bot automatically.
